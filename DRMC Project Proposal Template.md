## **13. Mid-Project Progress Update Readout**

*(This section is to be completed at regular intervals during the project execution phase, as determined by the Executive/Commodore. Use a separate sheet or log for ongoing updates.)*

The purpose of this readout is to provide a snapshot of the project's health against its primary metrics (**Budget**, **Timeline**, and **Scope**) while it's being executed.

| Readout Date | Current Phase/Progress | Budget Status | Timeline Status | Scope Changes (Yes/No) | Key Issue/Action Required |
|--------------|------------------------|---------------|-----------------|------------------------|---------------------------|
| Example: 2026-03-15 | 50% of Phase 1 Complete | $10,000 spent (5% under budget) | 2 weeks ahead of schedule | No | Need to finalize contractor for Phase 2. |
|              |                        |               |                 |                        |                           |

- **Budget Status:** State the current expenditure and percentage variance against the approved budget for the current phase/overall project.
- **Timeline Status:** State the current progress against the schedule (e.g., % complete, weeks ahead/behind schedule).
- **Scope Changes:** Describe any approved changes to the original scope and their impact on cost or time.
